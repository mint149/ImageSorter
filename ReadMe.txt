ImageSorterは画像整理プログラムです．

使い方:
1．Ctrl+Oで整理したい画像が入ったフォルダを開く
2．名前順で最初に来る画像が開くので，DFJKのどれかを押す．すると各キーに割り当てられたフォルダに画像が移動．
3．ひたすら2を繰り返し，空の画像が出てきたら整理完了．
