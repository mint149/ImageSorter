プロコン2013年度プログラム(エンコーダ)v16

はじめに:
競技部門で使用するエンコーダです。
デコーダはまだ未完成なのであまり意味はありませんが、暗号化したらどうなるのかの検証にどうぞ。
あと、サイコロの画像は自由に使ってください。
バグ報告、アイデア待ってます！

使い方:
1.開くボタンを押して、問題文の書かれたテキストファイル(サンプルとしてstring.txtを添付)を開きます。
2.コンボボックスを押すと、エンコード方法が選べます。
3.エンコードボタンを押すと、パケットの画像が表示されます。

ショートカット一覧:
Ctrl+O…ファイルを開く
Ctrl+E…エンコード開始
Ctrl+↑…エンコード方法を一つ上にずらす
Ctrl+↓…エンコード方法を一つ下にずらす
Ctrl+←…前のパケット画像を表示
Ctrl+→…次のパケット画像を表示
